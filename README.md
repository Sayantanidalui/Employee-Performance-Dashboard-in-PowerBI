# Employee-Performance-Dashboard-in-PowerBI

## 📌 Project Overview
The **Employee Performance Dashboard** is an interactive HR analytics dashboard built using **Power BI**. It analyzes employee productivity, performance trends, and attrition patterns to help organizations make data-driven HR decisions. And implementation of an unique feature - **Dark mode & Light mode toggle**

---

## 🛠 Tools & Technologies
- Power BI
- DAX (Data Analysis Expressions)
- Data Modeling
- Interactive Visualizations

---

## 📊 Dashboard Pages

### 1️⃣ Employee Overview
This page provides a high-level summary of the workforce including employee distribution, average income, and years at the company.

**Insights**
- Total Employees: **1480**
- Average Income: **6.5K**
- Average Years at Company: **7**
- **Research & Development** has the highest number of employees (967)
- Workforce gender distribution: **60% Male and 40% Female**

---

### 2️⃣ Performance Overview
This section analyzes employee productivity, training impact, workload, and performance ratings.

**Insights**
- Total Projects Handled: **6668**
- Average Weekly Working Hours: **50 hours**
- **Senior employees** contribute the highest number of projects (~2.4K)
- Employees who attend **more training sessions show slightly higher performance ratings**
- The relationship between absenteeism and performance rating appears inconsistent, suggesting that absence days alone may not strongly determine employee performance.

---

### 3️⃣ Attrition Overview
This page focuses on employee turnover patterns and factors influencing attrition.

**Insights**
- Attrition Count: **238 employees**
- Attrition Rate: **16.08%**
- Highest attrition occurs in **Research & Development**
- Employees in **lower salary slabs (up to 5K)** show the highest attrition
- Most employees leave within the **early years of employment**

---

## 🚀 Key Features
- Interactive Power BI Dashboard
- **Dark Mode / Light Mode Toggl**e
- Department and Role Level Analysis
- Training vs Performance Analysis
- Attrition Trend Analysis
- Dynamic Slicers and Filters
